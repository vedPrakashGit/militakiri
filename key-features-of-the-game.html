Key features of the Game

Militakiri is easy to learn but hard to master.

Simple design of the pieces to demonstrate the properties.

The board game is designed for modular play with single or double board.

A pawn, rank and tower can conquer each other.

Endgame rule, point system, rank system and Elo rating.

Promote from a rank to a tower.

Using spare towers in the game.

Hitting increases the rank.

Players: 2

Game duration per player:

10 minutes - 120 minutes for single board

20 minutes - 240 minutes for double board

24 hours/move - 72 hours/move for correspondence, single and double board

Game duration via time clocks

Game type: abstract tactical board game

all the opponent’s towers off the board
